## 2. Layer Mapping Intelligence Scheme

To prepare the platform for future ML and AI integrations without bloating the codebase with unsolicited complex engines, we introduce structural metadata tracking fields on `dxf_layer_mappings`.

### A. New Fields on `dxf_layer_mappings`
* **`suggested_type`** (`VARCHAR(40)`): Autogenerated baseline classification guess based on native DXF layer nomenclatures. Default value represents `'UNKNOWN'`.
* **`confidence_score`** (`INT`): Representation score ranging from `0` to `100` detailing structural classification matching integrity.
* **`mapping_source`** (`VARCHAR(20)`): Audits the driver origin of the active assignment. Enums:
  - `SYSTEM`: Mapped via initial string pattern matching rules.
  - `TEMPLATE`: Auto-loaded from a reusable layout profile.
  - `USER`: Manually revised or overridden by the draft operations team.

### B. Adaptive Evaluation Protocol (Rule-Based Heuristic)
The system evaluates discovered layer name matches and assigns recommendations via deterministic sub-string searches:
* **`PLOT`**: Matches `PLOT`, `PLT`, `PARCEL`, `SEC_` (Confidence: `90`, Source: `SYSTEM`).
* **`ROAD`**: Matches `ROAD`, `STREET`, `WAY`, `CIRCULATION`, `AVE` (Confidence: `85`, Source: `SYSTEM`).
* **`AMENITY`**: Matches `PARK`, `GARDEN`, `GREEN`, `AMENITY`, `CIVIC` (Confidence: `80`, Source: `SYSTEM`).
* **`UTILITY`**: Matches `UTIL`, `LIGHT`, `WATER`, `ELECTRIC`, `DRAIN`, `SEWER` (Confidence: `85`, Source: `SYSTEM`).
* **`BOUNDARY`**: Matches `BOUND`, `BORDER`, `PERIMETER`, `LIMIT` (Confidence: `95`, Source: `SYSTEM`).
* **Unknowns**: Fallback values are set to `UNKNOWN` with a Confidence Score of `0` and mapping source `SYSTEM`.

## 4. Visual Studio Interactive Workflow

The Layer Mapping Studio UI comprises five core layout sections designed around real-time desktop efficiency:

1. **Upload Discovery Stream**: Initial dropzone is verified, executing asynchronous parsing logs.
2. **Template Profiles Drawer**: Allows cloning, deleting, saving, or applying customized survey maps mappings.
3. **Smart Review Board**: Dense tabular array displaying discovered CAD layers alongside their native group colors, entity counts, intelligent recommendations, confidence scores, and current user mapped assignments.
4. **Interactive Validation Panel**: Real-time client-side compiler highlighting warnings in case of missing required layers (`PLOT`, `ROAD`, `BOUNDARY`) or structural duplicates.
5. **Import Dry-Run Preview**: High-visibility summary card representing totals, mapped classifications counts, active warnings, and transaction metadata.
